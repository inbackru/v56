# Инструкция по настройке Telegram Bot для InBack

## Проблема
Telegram Bot получает ошибку "Unauthorized" при отправке сообщений.

## Решение

### 1. Проверка бота через @BotFather
1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/mybots`
3. Выберите бота @inbackbot
4. Убедитесь, что бот активен

### 2. Получение нового токена
1. У @BotFather отправьте `/mybots`
2. Выберите @inbackbot
3. Нажмите "API Token"
4. Скопируйте новый токен

### 3. Обновление токена в Replit
1. Откройте Secrets в Replit
2. Найдите TELEGRAM_BOT_TOKEN
3. Обновите значение новым токеном

### 4. Проверка бота
1. Найдите @inbackbot в Telegram
2. Отправьте `/start`
3. Убедитесь, что бот отвечает

### 5. Связывание аккаунта пользователя
Для получения уведомлений пользователь должен:
1. Написать @inbackbot
2. Отправить `/start`  
3. Отправить `/link email@example.com`

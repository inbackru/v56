Как это работает:
• Система находит ЖК без nearby данных (добавленные парсером)
• Получает информацию из OpenStreetMap API (~10-20 сек на ЖК)
• Сохраняет данные в БД
• Пользователи видят актуальную информацию на сайте

CLI команды

Для обновления через командную строку:

# Обновить 5 ЖК
python3 nearby_auto_updater.py

# Обновить все ЖК требующие обновления
python3 nearby_auto_updater.py --all

Настроить автоматическое обновление (cron):

# Каждые 30 минут проверять и обновлять
*/30 * * * * cd /path/to/project && python3 nearby_auto_updater.py
